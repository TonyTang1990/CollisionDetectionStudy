# CollisionDetectionStudy
碰撞检测实战学习

## 工具

1. Unity版本(2019 4.31f1c1)
2. Visual Studio 2019

## 最终目标

通过深入学习碰撞检测里的数学知识，实战实现一套常规碰撞检测里需要用到的碰撞检测系统，为未来技能系统的碰撞判定打下基础。
